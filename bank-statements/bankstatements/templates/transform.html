Bank Statement Transformer

This web application is used to send a request to Azure to instigate the transformation of PDF Bank Statements into Excels.

All folders within 'Documents > Logic App Testing > Bank Statements' will be transformed.
The data from within the same folder will be concatenated together into one Excel, so keep statements relating to different accounts in their own seperate folder.
